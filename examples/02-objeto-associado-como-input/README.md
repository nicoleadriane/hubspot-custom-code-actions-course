# 02 - Objeto associado como input

Exemplo para reforçar que dados de objetos associados podem entrar como data source no workflow e chegar no código como input.

## Cenário

Um deal inscrito precisa usar dados da empresa associada.

## Inputs esperados

- `dealname`
- `associated_company_domain`
- `associated_company_segment`

Esses campos devem ser configurados no workflow usando data source de empresa associada.
