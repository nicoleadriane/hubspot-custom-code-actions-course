# Glossário

## Custom Code Action

Ação de workflow que executa JavaScript ou Python em ambiente serverless gerenciado pelo HubSpot.

## Input field

Valor disponibilizado para o código via `event.inputFields`.

## Data source

Fonte de dados adicionada ao workflow para usar propriedades de outros registros ou outputs de ações anteriores.

## Output field

Valor retornado pela ação para ser usado em etapas seguintes do workflow.

## Correlation ID

Identificador usado para rastrear uma execução entre logs, outputs e sistemas externos.

## Retryable

Erro que pode ser tentado novamente sem risco de duplicar efeitos.

## Idempotência

Propriedade de uma ação que pode ser executada mais de uma vez gerando o mesmo resultado final.
